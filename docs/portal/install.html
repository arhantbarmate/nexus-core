Node Setup

Deployment · Local Hardware · Zero-Trust
MOD_00 // PRE_FLIGHT_CHECK

Prerequisites

Ensure the following environment variables and dependencies are active before initializing the sovereign stack.

  • Python 3.10+ (System Path Enabled)
  • Git Version Control
  • Ngrok Account (For Zero-Cost Tunneling)
  • Flutter SDK 3.38.6 (Only if compiling Body from source)
MOD_01 // INITIALIZATION

Clone & Environment

Initialize the repository and isolate the Python environment to prevent dependency conflicts.

# Clone the repository
git clone https://github.com/arhantbarmate/nexus-core.git
cd nexus-core

# Setup Virtual Environment (Windows)
python -m venv venv
venv\Scripts\activate

# Install Core Dependencies
pip install -r requirements.txt
MOD_02 // LIFECYCLE_MANAGEMENT

Automated Node Control

Use the specialized batch scripts to manage the lifecycle of the Sentry, Brain, and Tunnel layers.

Start Sequence: Initializes FastAPI Brain & Ngrok Tunnel.

./start_nexus.bat

Stop Sequence: Flushes WAL journal and terminates processes.

./stop_nexus.bat
MOD_03 // EXECUTION_SURFACE

Client Compilation

If building from source, the Flutter Body must be compiled with the correct base-href to align with the Gateway architecture.

cd client
flutter build web --release --base-href "/nexus-core/app/" --no-tree-shake-icons

Verification Checklist
